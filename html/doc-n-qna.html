Expliquer la différence entre une requête HTTP synchrone et asynchrone. :

Une requête synchrone bloque le client jusqu'à la fin de l'opération tandis qu'une requête asynchrone ne bloque pas le client, c'est-à-dire que le navigateur est réactif
